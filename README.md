## 🚀 Features
- Light & Dark mode with localStorage persistence  
- Responsive grid layout for all screen sizes  
- Platform‑themed cards (Facebook, Instagram, Twitter, YouTube)  
- Follower counts and daily overview metrics  
- Clean, modern UI with smooth transitions  

---

## 🛠️ Technologies Used
- **HTML5**  
- **CSS3**
- **JavaScript**
- **Google Fonts**  
- **Font Awesome Icons**

---

## 📱 Responsive Design
The layout adapts from 4‑column → 2‑column → 1‑column depending on screen width.
